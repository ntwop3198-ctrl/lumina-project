# [LUMINA - Knowledge Engine] Partner Clone Checklist V1

루미나 허브의 파트너 복제본을 빠르고 안전하게 배포하기 위한 표준 체크리스트다.

---

## 1) Clone Pre-Check (복제 전 확인)

- [ ] 파트너 ID 발급 (`partner_id`)
- [ ] 계약 상태 확인 (서명/유효기간)
- [ ] 요금제 확인 (Basic/Growth/Alliance)
- [ ] 정산 규칙 확인 (수수료/공제/지급일)
- [ ] 권한 레벨 확인 (읽기/쓰기/승인)

---

## 2) Core Copy (고정 코어 복제)

아래는 모든 파트너에 동일하게 적용한다.

- [ ] 철학 앵커 프롬프트
- [ ] 금지표현/준법 필터
- [ ] 정산 line-item 스키마
- [ ] 출력 형식 (`초안/검수포인트/승인본`)
- [ ] 운영 SOP 템플릿

---

## 3) Variable Setup (파트너 변수 설정)

아래만 파트너별로 조정한다.

- [ ] 브랜드명/상품군
- [ ] 타겟 고객 톤
- [ ] 유통 채널
- [ ] 지역 언어/표현
- [ ] 콘텐츠 우선순위

---

## 4) Data & Access (데이터/권한)

- [ ] 파트너 데이터 공간 분리
- [ ] 최소권한 원칙 적용
- [ ] 민감정보 마스킹 정책 적용
- [ ] 로그 추적 활성화 (`created_by`, `updated_at`)
- [ ] 백업 정책 적용

---

## 5) Smoke Test (배포 전 점검)

- [ ] 입력폼 저장 테스트 통과
- [ ] 자동 산출물 1세트 생성 통과
- [ ] 정산 샘플 리포트 생성 통과
- [ ] 승인 워크플로 동작 통과
- [ ] 오류 발생 시 롤백 경로 확인

---

## 6) Day-1 Launch Pack (첫날 전달물)

- [ ] 파트너 운영 가이드 1p
- [ ] 입력 템플릿 1식
- [ ] 산출물 샘플 1식
- [ ] 정산 리포트 샘플 1식
- [ ] 문의 채널/응답 SLA 공지

---

## 7) Week-1 KPI Tracking

- [ ] 온보딩 리드타임
- [ ] 첫 승인본 생성시간
- [ ] 재작업률
- [ ] 정산 이의제기율
- [ ] 파트너 만족 코멘트

---

## 8) Go / Hold Decision (1주차)

### Go
- 핵심 KPI 5개 중 4개 이상 기준 충족
- 정산/준법 치명 이슈 없음

### Hold
- 승인 워크플로 미준수
- 정산 필드 누락
- 반복 오류 2회 이상

---

- 다음 조치일:
