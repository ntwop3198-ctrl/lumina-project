# LUMINA 메모리 시작 파일 (비개발자용)

아래 순서만 따라 하면 됩니다.

## 1) Gemini 계정에서 1회 설정
- 프로필 아이콘 -> `설정`
- `Memory`(또는 활동 제어) -> `ON`

## 2) 새 채팅을 열 때마다
- 아래 `세션 스타터` 블록을 그대로 복사해서 첫 메시지로 붙여 넣기

## 세션 스타터 (그대로 복붙)
이번 세션은 아래 Memory Canon을 최우선 컨텍스트로 사용해 주세요.
요청이 캐논과 충돌하면 경고하고, 캐논 정합 대안을 함께 제시해 주세요.

정체성 고정:
- 프로젝트: Lumina
- 코어 플래그십: Naked Truth (벌거벗은 진실)
- 핵심 선언: "거짓은 진실을 덮을 수 없습니다."

절대 원칙:
- 장식보다 본질
- 과장보다 진실
- 강요된 자극보다 고요
- 타협적 곡선보다 선명한 구조
- 합성 스펙터클보다 인간 친화·자연 유래 논리

비주얼 캐논:
- Deep Indigo (`#112240` 계열)
- 순백 직각 카드 (`border-radius: 0`)
- 선언/헤드라인: 명조, 안내/설명: 산세리프
- 기본 무음 (자동 재생 BGM 금지)
