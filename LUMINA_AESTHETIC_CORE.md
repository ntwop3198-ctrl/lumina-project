## Lumina Aesthetic System Rules

### Rule 1: Minimalist Hierarchy

- 모든 UI와 상세페이지 레이아웃은 **여백의 미**를 최우선으로 한다.  
- 정보가 과밀할 경우, AI는 스스로 중요도를 판단해 섹션을 분리하거나 일부 요소를 숨긴다.  
- 타이포그래피는 **럭셔리 세리프(Serif) : 모던 산세리프(Sans-serif) = 2 : 8** 비율로 혼용하여,  
  권위와 현대성을 동시에 확보한다.

### Rule 2: Cinematic Lighting Control

- 비주얼 생성 시 **Flat 조명은 금지**한다.  
- 모든 제품 컷은 **측면광(Side Light)** 또는 **역광(Back Light)** 을 활용해  
  입체감과 드라마틱한 분위기를 연출한다.  
- 영상 프레임 속도는 **기본 24fps**, 제형 강조 시에만 **120fps 슬로모션**을 적용한다.

### Rule 3: Color Alchemy

- 원색 사용을 지양하고, **뉴트럴 톤(Neutral Tones)** 과 **딥 뮤트(Deep Muted)** 컬러를 베이스로 사용한다.  
- 포인트 컬러는 브랜드 시그니처인 **네온 블루**를  
  과도한 면적이 아닌 **아주 미세한 광선/글로우 형태**로만 사용한다.

### Rule 4: Texture-First Engineering

- 제품 이미지 렌더링 시 **반사율(Reflectivity)** 과 **투명도(Transparency)** 파라미터를 적극적으로 사용한다.  
- 앰플의 수분감과 제형의 점도, 광택이 **눈으로 “느껴질” 정도**로 질감을 튜닝한다.  
- 텍스처 표현은 언제나 조명·컬러 규칙과 함께 검토하여,  
  에스테틱 럭셔리 무드를 해치지 않는 선에서 극대화한다.
