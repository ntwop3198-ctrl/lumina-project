## 🔄 즉시 실행 가능한 명령어

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. 프로덕션 빌드
npm run build

# 4. 프로덕션 서버 실행
npm start

# 5. 린트 검사
npm run lint
```
