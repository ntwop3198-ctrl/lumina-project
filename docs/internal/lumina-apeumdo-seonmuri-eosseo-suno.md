## Title (Suno 제목란)

```
아픔도 선물이었어
```

## Style (Suno 스타일란)

```
Korean ballad, emotional male vocal,
sparse acoustic guitar opening, gentle strings building through chorus,
bittersweet and liberating, reflective and warm,
not epic, 70~80bpm,
strong strings swell on final chorus, fade out ending
```

> 혼인정리·실명·호텔 맥락은 **스타일란에 넣지 않음**. `(fade out)` 은 Style 끝에만 — **가사란에는 넣지 않음** (Suno 코러스 중복 유발).

## Lyrics (Suno 가사란 — 붙여넣기용 · v1 정본)

```
[Intro]
(잔잔한 어쿠스틱 기타)

[Verse 1]
다신 사랑 안 해 울부짖던 그 밤에
세상이 미웠고 하늘을 원망했어
근데 있잖아 지나고 보니 알겠어
그건 벌이 아니라 선물이었다는 걸

[Verse 2]
매일 고요한 저녁이면 잠을 들 수가 없었어
언제부터인가 술이 가장 친근한 벗이 되었더라
이불 속에서 눈물로 베개를 적셨던 적이
헤아릴 수도 없이 많아

[Chorus]
고마워 아프게 해줘서
고마워 떠나가 줘서
전생에 진 빚을 이렇게 갚게 해줘서
이제야 자유롭다 이제야 가볍다
다신 사랑 안 해가 고마워로 바뀌었어

[Verse 3]
저항하면 이자가 붙는다 했잖아
또 만나고 또 아파야 한다 했잖아
이번엔 달라 담담하게 받을게
잘 왔다 이제 끝내자 잘 가라

[Final Chorus]
고마워 아프게 해줘서
고마워 떠나가 줘서
전생에 진 빚을 이렇게 갚게 해줘서
이제야 자유롭다 이제야 가볍다
다신 사랑 안 해가 고마워로 바뀌었어

[Outro]
사랑은 빚이었고
이별은 청산이었고
아픔은 수행이었어
고마워 고마워 진심으로 고마워
```
