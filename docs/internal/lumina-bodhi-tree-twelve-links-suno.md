## Lyrics (Suno 가사란 — 붙여넣기용 · v1)

```
[Intro]
(잔잔한 피아노·현, 보리수 바람)

보리수 아래 홀로 앉아
눈을 감고 세상을 보니
바람도 없고 소리도 없는
그 자리에서 모든 게 보이네

[Verse 1]
무명(無明)이란 어둠 속에서
행(行)이 일고 식(識)이 깨어나
명색(名色)으로 모습을 갖추고
육처(六處)로 세상을 만지네
눈으로 보고 귀로 듣고
코로 맡고 혀로 맛보며
몸으로 느끼고 마음으로 생각하니
이 모든 게 인연으로 이어지네

[Verse 2]
촉(觸)이 닿아 수(受)가 일어나
애(愛)가 타오르고 취(取)가 쌓여
유(有)가 되어 생(生)을 받으니
노사(老死)는 피할 수 없구나
잡으려 할수록 더 멀어지고
버리려 할수록 더 달라붙어
이 사슬의 끝을 찾아보니
시작도 없고 끝도 없더라

[Chorus]
이 모든 것 연기(緣起)로 왔다가
연기로 돌아가리니
집착을 놓으면 괴로움 사라지고
텅 빈 곳에 자유가 오네

[Bridge]
무명이 사라지면 행이 멈추고
행이 멈추면 식이 고요해져
하나씩 하나씩 꺼져가는 불꽃
그것이 바로 해탈이더라
억지로 끄려 하지 않아도
인연이 다하면 저절로 꺼지고
구하지 않아도 때가 되면
텅 빈 그 자리에 고요가 오네

[Final Chorus]
이 모든 것 연기(緣起)로 왔다가
연기로 돌아가리니
집착을 놓으면 괴로움 사라지고
텅 빈 곳에 자유가 오네

[Outro]
보리수 아래 바람이 불고
꽃잎 하나 조용히 지네
왔다가 가는 것이 삶이고
가고 나서 또 오는 것이 삶이네
(fade out)
```

## 十二緣起 (가사 대응)

| 순연기 (Verse) | 역연기 (Bridge) |
|----------------|-----------------|
| 1 무명(無明) | 무명 사라지면 → |
| 2 행(行) | 행 멈춤 → |
| 3 식(識) | 식 고요 → |
| 4 명색(名色) | (Bridge는 핵심 역순) |
| 5 육처(六處) | |
| 6~11 촉→수→애→취→유→생 | |
| 12 노사(老死) | 해탈 · 고요 |
