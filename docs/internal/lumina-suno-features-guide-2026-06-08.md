## 상단 탭 (Library / Home)

| 탭 (UI) | 기능 |
|---------|------|
| **노래 (Songs)** | 내가 만든 곡 **목록** |
| **플레이리스트** | 곡 **모아두기** |
| **작업 공간 (Workspace)** | **작업 중**인 곡 |
| **목소리 (Voices)** | 보컬 **스타일·톤** 설정 |
| **표지 그림 (Cover Art)** | 앨범 **커버 이미지** |
| **갈고리 (Hooks)** | **후크 멜로디** 추출 |
| **홈스를 좋아했어요 (Liked)** | **좋아요** 누른 곡 |
| **역사 (History)** | **작업 기록** |

## 곡 상세 · ··· 더보기 (재확인)

| 메뉴 | 설명 | 권장 |
|------|------|------|
| **게시** | Suno 플랫폼 공개 | ✅ |
| **업로드** | DistroKid 등 **배포** | 발매 시 |
| **생성하다** | 이 곡 기반 **새 버전** | A/B 테스트 |
| **생기 있게 하다** | **음질 향상** | 소리만 아쉬울 때 |
| **댓글 허용** | 타 사용자 댓글 | ✅ ON |
| **리믹스 허용** | 타 사용자 리믹스 | ✅ ON (씨 확산) |
| **프로필 고정** | 프로필 상단 노출 | 대표곡 1~2 |
| **캡션·가사·스타일 편집** | 메타 수정 | 배포 전 |

→ 요약: `lumina-suno-library-publish-guide.md`

## Suno Studio (미션 팝업) — **지금 안 해도 됨**

생성 후 **「스튜디오에 입장하세요」** 등 **9개 미션 · 6일** 안내 = **유료 추가 기능** 유도.

| 미션 | 기능 | 비고 |
|------|------|------|
| 스튜디오 입장 | Suno Studio 열기 | **유료** |
| 겹겹이 쌓기 | 보컬·악기 **레이어** | **유료** |
| MIDI 변환 | **악보** 추출 | **유료** |
| 분리 | 보컬·드럼·베이스 **스템** | **유료** |

**지금 할 일:** 팝업 **닫기** → 생성된 **2버전 들어보기** → 마음에 드는 버전 **게시**.

TikTok ₩90k 미션과 같이 — **稳**: 소액·듣기·게시 먼저.
