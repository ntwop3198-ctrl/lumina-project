## Lyrics (Suno 가사란 — 붙여넣기용 · v1)

```
[Intro]
(잔잔한 피아노, 공장 불빛·도시 밤 잔향)

[Verse 1]
동관 공장 불빛 아래 땀 흘리던 날
심천 아파트 창문엔 꿈이 가득했지
코로나 바람 한 번에 다 쓸려가고
빈손으로 부천 골목 다시 서는 날

[Verse 2]
이혼 서류 도장 찍고 혼자 남았지
12살 공주 손 잡고 새벽을 걸었어
붕소님 앞에 향 하나 피워 올리며
오늘도 나나 불빛 아래 웃어보네

[Hook]
남매야 남매야 南妹야
넘어져도 다시 서는 여자
동관도 심천도 다 꿈이었어도
부천 이 골목이 내 땅이야

[Bridge]
잃은 것보다 남은 것 세어보면
공주 하나 — 그것으로 충분해
붕소님도 알고 있어 내 마음을
비워진 그릇 다시 채워지는 것을

[Outro]
남매야 남매야 南妹야
오늘도 불빛 아래 살아있어
(fade out)
```
