# 38선 너머 (三八线那边 · Beyond the 38th)

> **映山红 결** · 가야금·해금 최소 반주 · 이분가족 상봉의 애절한 한(恨)  
> Cyberpunk EDM 「38선」과 **같은 주제, 다른 결** — Suno Short 붙여넣기용  
> **德友**가 추천한 映山红(진달래) 정서 × 남편이 아내·자녀를 그리는 심정

## Style

```
Gugak Folk, Gayageum, Haegeum, slow tempo, 60~70bpm, minimal instrumentation, melancholic, sorrowful, Korean-Chinese fusion, gender-neutral vocal, haunting, sparse, heartbreaking, not epic, trilingual, intimate, stillness
```

## 메모

| | Cyberpunk 「38선」 | 이곡 38선 너머 |
|--|-------------------|----------------|
| **Style** | Future bass, 130bpm, dystopian | Gayageum·Haegeum, 60~70bpm, sparse |
| **정서** | 부수고·외치고·저항 | 품고·그리워하고·말없이 울다 |
| **주인공** | 세대·알고리즘·데이터 | 남편 → 북쪽 아내·孩儿们 |
| **공통** | **선(線)이 몸·가족을 가른다** · **三八線 / 삼팔선** |
| **관계** | **求死** — 틀 파괴 | **온(穩)** — 증언·체념 사이의 한 |

- **映山红** — 외치지 않고 꽃 하나로 말하는 결; 혁명가를 애절·처량하게 재해석한 **德友** 추천곡에서 영감
- **핵심 장면** — 다시 못 볼 걸 알면서 부둥켜 안고 통곡; 「真不想说，我走了那句话」
- Suno **잘림 주의** — trilingual이 길면 Verse당 언어 하나만 남기고 시도
