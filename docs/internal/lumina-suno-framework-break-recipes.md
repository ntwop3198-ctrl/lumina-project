## Cursor 추가 레시피 5 — 「없다」까지

### 1. 역사×K-pop (北+東)
```
Style: K-pop girl group chorus, Joseon court gayageum, trap hi-hats, 110bpm, polished, ironic, not epic
Lyrics: 세종대왕 찬가 1절만 — 한국어
```
**느낌:** 아이돌 후렴인데 가야금·한글 창제 서사 — **기괴한 정당성**

### 2. 마사지×Ambient (南+下)
```
Style: spa ambient, soft pads, 60bpm, whisper vocal, ASMR texture, healing, no beat drop
Lyrics: 봉향씨 「아픈 만큼 가져간다」 — 3줄만
```
**느낌:** 클럽이 아니라 **손끝 소리** 같은 노래

### 3. 트로트×Philosophy (西+北)
```
Style: Korean trot, accordion, bouncy rhythm, 130bpm, comedic brass, male vocal, not epic
Lyrics: 구소 「작은 방 한 칸이면 충분해」 + 德友 「可怜之人必有可恨之处」
```
**느낌:** 뽕짝인데 **불자 철학** — 사우나 아저씨들 반응용

### 4. Cui Jian×Classical (東+北)
```
Style: Chinese root rock, string quartet, electric guitar, trumpet, 92bpm, raw, not epic
Lyrics: 가시내 후렴 KO+ZH 각 2줄
```
**느낌:** 花房姑娘 뿌리 + **현악四重奏** — 이미 록이었던 걸 **한 겹 더 깸**

### 5. Silence Break (下 → 空)
```
Style: near silence, single haegeum note, room ambience, 70bpm, sparse, meditative, not epic
Lyrics: [Intro] 求空 [Verse] (침묵 — 가사 없음 8마디) [Chorus] 「작은 것이 전부야」 한 줄만
```
**느낌:** Suno가 헷갈리면 **성공** — 형식이 **없다**에 가까워짐

## 실전 규칙 (Suno 역이용)

1. **태그 5~8개** — 너무 많으면 Suno가 **하나만** 고름 → **의도적 충돌 2~3쌍**만
2. **not epic** 항상 — 틀 깨도 **Lumina 준법**
3. **가사 Short** — 잘리면 **파괴의 절반**만 남음
4. **1곡 1실험** — 사면팔방은 **8곡에 나눠** — 집착하지 않기
5. **본질 한 줄** — 실험 후 「이 한 줄만 남는가?」 — 求己 검증
