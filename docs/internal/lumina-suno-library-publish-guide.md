## 탭/버튼 — 무엇을 누를까?

| 메뉴 | 기능 | **권장** |
|------|------|----------|
| **게시 (Publish)** | Suno 플랫폼에 **공개** | ✅ 완성곡·마음에 드는 버전 → **먼저 게시** |
| **업로드 (Upload)** | DistroKid·Spotify 등 **배포 연동** | ✅ **정식 발매**할 때 (세종대왕찬가처럼 DistroKid 우선) |
| **생성하다 (Create/Extend)** | 이 곡 기반 **새 버전** (스타일·가사 변경) | 반응 보며 A/B 테스트할 때 |
| **생기 있게 하다 (Enhance)** | 음질·생동감 **리마스터** | 마음에 드는데 소리만 아쉬울 때 |
| **댓글 허용** | 다른 사용자 댓글 | ✅ **켜기** — 피드백·노출 |
| **리믹스 허용** | 타 사용자가 **리믹스** 가능 | ✅ **켜기** — 씨 넓게 퍼짐 (광진 결정) |
| **노래를 프로필에 고정** | 프로필 상단 **고정** | 대표곡 정할 때만 (1~2곡) |
| **캡션·가사·스타일 편집** | 메타데이터 수정 | 배포 전 제목·가사 최종 점검 |

---

## 추천 순서 (稳准快)

```
1. Suno에서 만족하는 버전 확정
2. 게시 (Suno 공개) + 댓글·리믹스 허용 ON
3. TikTok 등 SNS 클립·홍보
4. DistroKid 등으로 업로드 (TuneCore Suno 이슈 시 DistroKid 우선)
```

- **TuneCore:** Suno/Flow Music 미승인 이슈 → DistroKid 전례 활용  
- **업로드**는 “지금 당장”이 아니라 **발매 결정 후**
