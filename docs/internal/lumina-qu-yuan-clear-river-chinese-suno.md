## Lyrics (Suno · v1 正本)

```
[Intro]
(古琴低鸣，如冷冽江水流淌)

[Verse 1]
举世皆浊我独清
众人皆醉我独醒
谁人知晓这清醒的孤寒
唯有汨罗江水接我泪痕
愈攀高处愈感孤独
愈是清澈愈是独行
莫非是这世道变浑浊
还是我太过清醒

[Verse 2]
渔父问我为何如此执着
随波逐流便不再孤单
我岂不知 只是怕这身子沾染污浊
宁将清白付与这滔滔江水
宁折不弯的世道里
他选择了站立到最后
未曾胜利 却也未曾认败
那名字千年回响不止

[Chorus]
若稍稍低头便可少些痛苦
若稍稍闭眼便可展颜而笑
太过清澈而败下阵来的人
太过清醒而化身江河的人
屈原啊 你独自承受了时代的浪潮

[Bridge]
世人不断指责于我
然我心中那面镜子从未破碎
不妥协 是我唯一的权力
不屈膝 是我最后的尊严
败，未必是真败
屈，未必是真活
那份执着 那份清醒 那份孤独
反而活过了千年

[Final Chorus]
若稍稍低头便可少些痛苦
若稍稍闭眼便可展颜而笑
太过清澈而败下阵来的人
太过清醒而化身江河的人
屈原啊 你独自承受了时代的浪潮

[Outro]
千年已过江水依然静静流淌
每逢端午投粽以慰你魂灵
曾经独清的你 如今化作江河安息吧
那颗太过清醒而孤独的心
至今仍活在某人的胸怀之中
(fade out)
```
