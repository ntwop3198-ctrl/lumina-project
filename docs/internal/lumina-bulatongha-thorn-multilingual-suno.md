# 부르하통하의 가시내 (화방 아가씨에게 · 다국어 통합)

> Suno 붙여넣기용 · 한국어 위주

## Style

```
Chinese root rock, folk rock, electric guitar riff, trumpet, earthy, serenade, mournful male vocal, mid tempo, 92bpm, raw, lyrical, nostalgic, river ambience, multilingual, Korean main vocals, not epic
```
