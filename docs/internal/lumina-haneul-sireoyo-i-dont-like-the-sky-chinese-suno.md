## Lyrics (Suno · v1 최종)

```
[Intro]
(轻柔的吉他琶音，温暖的大提琴)

[Verse 1]
说天空好的人都往上飞去了
我就待在这里 在地下
爬得高有什么好的
摔下来有多疼 我知道
空桶爬得越高
掉下来的时候越响越痛

[Verse 2]
我也摔过 摔得很重很大
触底之后才明白 还是地面好
天空不过是借来的速度
我的脚踩着的这片土地才是真的
看着地面才能捡到东西
光看天空只会挨饿

[Chorus]
我不喜欢天空 摔下来会痛
要看着地面 才能捡到东西
爬得高的空桶 摔下来声音响
我就喜欢地下
租金便宜 安安静静
根是从地下往深处扎的

[Verse 3]
也有过被冷风刮走的夜晚
但那个寒冷的夜晚成了我的根
因为受过伤 才变得坚强
因为跌倒过 才学会了走路
因为心空了 光才能进来
那不是恐惧

[Bridge - 哲学]
求小 — 小小的就够了
一缕阳光 一个微笑就够了
求苦 — 因为痛过所以明白
苦难让我变得更深沉
求己 — 答案终究在自己心里
在地下找到的
真空妙有 — 空了才能满
说没有 却又不是没有

[Final Chorus]
我不喜欢天空 摔下来会痛
要看着地面 才能捡到东西
爬得高的空桶 摔下来声音响
我就喜欢地下
空了才能真正地满
最低处才是最坚实的地方

[Outro]
我不喜欢天空
摔下来会痛 哈哈
就是喜欢地下
租金也便宜 哈哈哈
和地藏菩萨喝杯茶就够了
(fade out)
```
