# 일체유심조 (一切唯心造 · All Is Mind)

> **원곡동 3일 결실** · 境由心生，心由境造 · 15년 전 북경 광명이 숙소 청소하며 본 글귀  
> Cursor 정리 (Claude 대화 기반)

## Style

```
Gugak Folk, Gayageum, Haegeum, slow tempo, 65~75bpm,
minimal instrumentation, meditative, philosophical,
warm gender-neutral vocal, sparse, intimate,
Korean-Chinese-English trilingual, not epic
```
