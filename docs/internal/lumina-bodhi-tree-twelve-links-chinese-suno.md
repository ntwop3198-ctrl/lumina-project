## Lyrics (Suno · v1 正本)

```
[Intro]
(轻柔的钢琴与弦乐，菩提树下微风轻拂)

独坐菩提树下
闭目观照世间
无风无声无扰动
于那寂静处 万象皆明

[Verse 1]
无明之暗中
行起而识醒
以名色显其形
以六处触世界
以眼观 以耳听
以鼻嗅 以舌味
以身感 以心思
万事皆因缘相连

[Verse 2]
触至而受生起
爱燃而取积聚
因有而得生
老死终难逃
愈执愈远离
愈舍愈相随
寻这锁链之尽头
无始亦无终

[Chorus]
此一切由缘起而来
亦将循缘起而归
放下执着 苦即消散
空寂之处 自由自来

[Bridge]
无明灭则行止
行止则识寂
一点一点熄灭的火焰
那便是解脱
无需强行熄灭
缘尽自然灭
不求而时至
空寂处 静自来

[Final Chorus]
此一切由缘起而来
亦将循缘起而归
放下执着 苦即消散
空寂之处 自由自来

[Outro]
菩提树下风轻拂
一片花瓣悄然落
来而去者是人生
去而复来亦是生
(fade out)
```
