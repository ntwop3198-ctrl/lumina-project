# 진짜 사랑 (True Love)

---

## Title (Suno 제목란)

```
진짜 사랑 (True Love)
```

## Style (Suno 스타일란)

```
Emotional R&B, healing and introspective,
smooth groove, soft piano, warm bass,
soulful vocal, male or female,
75bpm, self-reflection, healing,
Korean lyrics, not epic
```

## Lyrics (Suno 가사란 — 붙여넣기용)

```
[Verse 1]
확인받고 싶어서 헤맸던 거야
사랑이 아니라 공허함 때문에
애인을 익숙한 자리로 밀어두고
새 자극에 흔들렸던 그 마음

[Verse 2]
들키지 않으면 괜찮다 속였지
죄책감보다 들킬까 봐 더 무서워
자기 가치를 남 반응으로 채우다
정작 곁에 있는 사람 잃어버렸어

[Pre-Chorus]
잘못인 거 알아
근데 왜 멈추지 못했을까
공허함을 사랑으로 채울 수 없어
그건 네가 해결해야 할 몫이야

[Chorus]
진짜 사랑은 확인이 아니야
옆에 있어도 설레는 사람
익숙해도 소중한 사람
그게 진짜야

[Bridge]
외롭고 허한 걸
연애 하나로 버티려 했던 너
이제는 스스로를 먼저 채워봐
그래야 진짜 사랑할 수 있어

[Outro]
자기 공허를 다루는 법을 배워
그때 비로소 한 사람을 사랑할 수 있어
진짜로
```
