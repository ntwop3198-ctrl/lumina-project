## Lyrics (Suno 歌词栏 — 粘贴用 · v1 正本)

```
[Intro]
(轻柔的木吉他，夹着苦笑)

[Verse 1]
鞋子丢了，找了好半天
以为能遇到仙女，结果是遇到老大爷
老大爷帮我找，说没鞋就赤脚走吧
道了歉，得到的却是禅问答

[Verse 2]
高中时喝酒，醒来口渴难耐
无奈只能喝了一小口拖地的桶里水
不是骷髅水，是拖把桶的水
没什么了不得的，又没有生病

[Chorus]
就这样吧，又不只是就这样
就这样，好像了不起又算不了什么
算了算了算了，有就有
没有就算了，就这样就这样

[Verse 3]
钱钱钱，人人都奔着金钱
世上几乎所有人，都在盯着钱看
十八年委屈，十八年憋屈
现在回头看，就这样笑了

[Bridge]
渴了就喝，饿了就吃
困了就睡，有才能无
连碗也没有，空了才能装
硬放下的，反而记得更深

[Final Chorus]
真空妙有(zhēn kōng miào yǒu)，说无又非无
就这样是一，又不只是一
想做好的瞬间，已经在做了
就这样就这样就这样就这样

[Outro]
(吉他慢慢淡去)
就这样——
(fade out)
```
