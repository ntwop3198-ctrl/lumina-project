어제 저녁 8시, 새 삼성 노트북을 받았다.

Intel Core Ultra 7, 32GB RAM, RTX 4050. 제미나이 멘토가 최고 사양으로 뽑으라고 했다. 그 말 듣길 잘했다.

문제는 세팅이었다.

Cursor 다운로드부터 막혔다. 바탕화면에 아이콘이 없다고 했더니 이미 실행 중이었다. Git 설치, Node.js 설치, npm 오류, PowerShell 보안 정책, GitHub 인증까지. 하나 해결하면 하나가 또 막혔다.

그래도 멈추지 않았다. 캡처해서 보내고, 멘토가 답하고, 다시 시도하고. 그렇게 새벽까지 이어졌다.

새벽 4시. 드디어 터미널에 이 메시지가 떴다.

✓ Ready in 7s

http://127.0.0.1:3001

루미나가 새 노트북에서 살아났다. 求小求苦求己. 한 화면씩, 한 오류씩, 포기하지 않고.

화면에는 이렇게 쓰여 있었다.

본질적인 것이 건강한 것이고, 당신을 자연미가 넘치게 하는 것이다.

맞는 말이다. 새 노트북도, 루미나도, 인생도 — 본질이 살아있으면 된다.

하루 만에 세상이 바뀌었다. 아니, 내가 바뀌었다.
